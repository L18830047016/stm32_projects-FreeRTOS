## 开发环境与编码说明

以上所有 Demo 均基于 **STM32F103C8T6 最小系统板** 开发与测试。开发过程中使用了以下工具与软件：

-   串口调试助手（用于日志打印）
    
-   Keil MDK5（工程编译与下载）
    
-   Visual Studio Code（代码编辑）
    

### 关于中文编码的说明

由于作者对 VS Code、Keil 以及串口工具中的中文编码机制尚未完全掌握，Demo 中的注释编码格式存在差异：

-   **Demo 1 ~ Demo 5**：采用 **UTF-8** 编码
    
-   **Demo 6 ~ Demo 18**：采用 **GBK** 编码
